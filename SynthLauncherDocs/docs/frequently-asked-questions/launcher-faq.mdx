# Launcher FAQ
## Is SynthLauncher a virus?
No, SynthLauncher has never had any sort of malware, including spyware, adware, or viruses. It does not have any and will not have any. We are a fully open-source project, so anybody can view our 
full source code on our [GitHub repository](https://github.cnom/SynthLauncher/SynthLauncher) any time. If you have any doubts, you can always check the code yourself. You can also submit the launcher to [VirusTotal](https://www.virustotal.com/gui/home/upload).
Please be aware of **false positives**, however.

## Does SynthLauncher support Minecraft: Bedrock Edition?
Unfortunately not at the moment, but there are plans for it to be added in the future. Currently, 
SynthLauncher only supports Minecraft: Java Edition
