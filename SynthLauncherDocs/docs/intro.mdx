# Introduction

## What is SynthLauncher?
SynthLauncher is a secure and open-source, Minecraft launcher with a sleek interface.\
The launcher is still ongoing development, but will feature;

- Support for offline accounts, without requiring the need for a Microsoft or Mojang account
- Full customization using custom CSS!
- A skin system supporting free capes
- Modpack support through CurseForge and Modrinth
- Mod loader support through Forge, Fabric, Quilt, and NeoForge
- Historic versions support (Alpha, Beta)
- A clean and intuitive UI

SynthLauncher was designed to be an alternative to other Minecraft Launchers

- SynthLauncher provides a safer alternative to TLauncher
- It provides support for offline accounts without the need for a Microsoft account, unlike Prism Launcher and MultiMC
- It provides the same offline functionality, skins support, and modpack support of SKlauncher, but with added stability.
